# Guardian Kids AI Site

Site estático oficial da extensão Guardian Kids AI, pronto para publicar com GitHub Pages e usar nos cadastros da Chrome Web Store, Mozilla Add-ons, Microsoft Edge Add-ons, Opera Add-ons e outras lojas.

## Estrutura

```text
guardian-kids-ai-site/
|-- index.html
|-- privacy.html
|-- support.html
|-- terms.html
|-- assets/
|   |-- logo.png
|   `-- screenshots/
|-- css/
|   `-- style.css
`-- README.md
```

## Publicar no GitHub Pages

1. Crie um repositório chamado `guardian-kids-ai-site` ou `guardian-kids-ai`.
2. Envie todos os arquivos desta pasta para a branch `main`.
3. No GitHub, abra `Settings -> Pages`.
4. Em `Source`, escolha `Deploy from a branch`.
5. Selecione `Branch: main` e `Folder: /root`.
6. Salve.
